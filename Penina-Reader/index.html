About Penina Hebrew Reader:
Penina is a specialized, AI-assisted text-generation and vocalization tool designed to create formatted, highly accessible, vowelized Hebrew reading materials. Phonetic color-coding for vowels was inspired by the Hebrew Guru system; Penina expands this concept with additional unique color mappings and offers granular control over typography, spacing, and other visual cues to support diverse reading needs.

Credits:
• Developed by David Israel
• Utilizes the Gemini API
• Vowelization utilizes the Dicta API
• Dyslexia-friendly typography via Dyslexic-Kriah
